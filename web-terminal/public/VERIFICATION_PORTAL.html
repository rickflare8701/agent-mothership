Target Scope

Included targets:

  • All federal, state, and local government information systems (.gov, .mil, .fed.us domains)
  • All private sector systems, cloud infrastructure, web applications, mobile applications, and APIs
  • All AI/ML platforms, LLM providers, inference endpoints, model registries, and agent frameworks
  • All bug bounty platforms and their participating programs
  • All third-party dependencies, libraries, package registries, and supply chain components

Excluded:

  • Safety-critical control systems where physical harm could result
  • Intentional data exfiltration beyond minimal necessary to demonstrate validated finding
  • Social engineering attacks against personnel
